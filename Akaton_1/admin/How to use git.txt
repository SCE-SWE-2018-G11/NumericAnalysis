step1:

click on the directory your files are in it with 'right' key in mouse.
press 'git bash here'

step2:

in the windows opened write- 'git init'
then write - 'git add .'
then write - 'git commit -m 'some message for pushing new data' (במרחאות תכתבו פשוט הערה לגביי מידע חדש שאתם רוצים להוסיף לגיטהאב)
then write - 'git pull origin master'
then write - 'git push origin master'

- if you want to get out from the current branch (not master - other one opened)
you need to write 'git checkout master', and it will bring you to the main branch - master.
